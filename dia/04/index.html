Egg Dropping Problem

Mayo 04

El problema de lanzamiento de huevos (Egg Dropping Problem) es un problema matemático en el que se tiene que determinar la altura máxima de un edificio desde la cual se puede lanzar un huevo sin que se rompa. Este problema se utiliza como ejemplo en el campo de la programación dinámica y la teoría de la complejidad computacional.

La solución al problema se encuentra utilizando un algoritmo de programación dinámica. La idea es ir probando lanzamientos desde distintas alturas y ver cuál es la altura máxima desde la cual se puede lanzar el huevo sin que se rompa. Para esto, se utiliza una matriz de n x m donde n es el número de huevos disponibles y m es el número de pisos del edificio.

En cada paso, se realiza un lanzamiento del huevo desde una altura determinada y se observa si se rompe o no. Si el huevo se rompe, se tiene que buscar la solución en los pisos inferiores con un huevo menos. Si el huevo no se rompe, se busca la solución en los pisos superiores con el mismo número de huevos disponibles.

El algoritmo se puede implementar de forma recursiva o iterativa. La implementación recursiva es más sencilla de entender, pero puede ser menos eficiente en términos de tiempo y espacio. La implementación iterativa es más eficiente, ya que utiliza una matriz para almacenar los resultados parciales y evita repetir cálculos.

En resumen, el algoritmo Egg Dropping Problem es un ejemplo clásico de la aplicación de la programación dinámica en la resolución de problemas complejos. Su solución se basa en la exploración sistemática de distintas opciones y la utilización de una matriz para almacenar los resultados parciales y evitar repetir cálculos.

Ejemplo

Supongamos que tenemos un edificio de 20 pisos y dos huevos disponibles. El objetivo es determinar la altura máxima desde la cual se puede lanzar un huevo sin que se rompa.

Para resolver este problema utilizando el algoritmo de programación dinámica, se tendría que crear una matriz de 2 x 20. En cada posición de la matriz, se almacenaría el número mínimo de lanzamientos necesarios para determinar la altura máxima desde la cual se puede lanzar el huevo sin que se rompa.

Supongamos que en la primera fila de la matriz se encuentran los resultados para un huevo disponible y en la segunda fila se encuentran los resultados para dos huevos disponibles.

En la posición (1,1) de la matriz, se tendría que lanzar el primer huevo desde el primer piso para poder determinar la altura máxima desde la cual se puede lanzar el segundo huevo sin que se rompa. Por lo tanto, el valor en esa posición sería 1.

En la posición (2,1) de la matriz, se tendría que realizar una búsqueda binaria para determinar la altura máxima desde la cual se puede lanzar el huevo sin que se rompa. Se lanzaría el primer huevo desde el piso medio (10) y se observaría si se rompe o no. Si se rompe, se tendría que buscar la solución en los pisos inferiores con un huevo menos. Si no se rompe, se buscaría la solución en los pisos superiores con el mismo número de huevos disponibles. En este caso, se tendría que lanzar el segundo huevo desde el piso medio de los pisos superiores (16) y se observaría si se rompe o no. Si se rompe, se tendría que buscar la solución en los pisos inferiores con un huevo menos. Si no se rompe, se tendría que buscar la solución en los pisos superiores con el mismo número de huevos disponibles. El valor en la posición (2,1) de la matriz sería igual al número de lanzamientos realizados para encontrar la solución.

En la posición (2,20) de la matriz, se tendría que realizar una búsqueda binaria para determinar la altura máxima desde la cual se puede lanzar el huevo sin que se rompa. En cada paso, se tendría que lanzar el primer huevo desde el piso medio del rango de pisos a explorar y se observaría si se rompe o no. Si se rompe, se tendría que buscar la solución en los pisos inferiores con un huevo menos. Si no se rompe, se tendría que buscar la solución en los pisos superiores con el mismo número de huevos disponibles. El valor en la posición (2,20) de la matriz sería igual al número de lanzamientos realizados para encontrar la solución.

En resumen, el algoritmo de programación dinámica para el problema de lanzamiento de huevos se basa en la exploración sistemática de distintas opciones y la utilización de una matriz para almacenar los resultados parciales y evitar repetir cálculos. En este ejemplo, se tendrían que realizar búsquedas binarias para determinar la altura máxima desde la cual se puede lanzar el huevo sin que se rompa.
